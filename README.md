# rock-paper-scissors

A web-based implementation of the classic rock-paper-scissors game. Play against the computer and enjoy a simple, interactive experience right in your browser.

## Contents

- **css/**: Styles for the game interface.
- **image/**: Visual assets for game elements.
- **js/**: JavaScript logic for gameplay.
- **index.html**: Main entry point for the web app.

## Features

- Interactive gameplay with visual feedback.
- Simple, intuitive interface for all ages.
- Live demo: [https://natnaeleyuel.github.io/rock-paper-scissors/](https://natnaeleyuel.github.io/rock-paper-scissors/)
- Built with HTML, CSS, and JavaScript.

## Usage

Clone the repository and open `index.html` in your browser to play the game:

```sh
git clone https://github.com/natnaeleyuel/rock-paper-scissors.git
```
